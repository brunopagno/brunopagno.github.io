Drread!

This project was created in (roughly) 48h for the Mini Jam 83: Dread

Game rules

You control a character named Dr.read

You can move around using the arrow buttons () around the protagonist

The objective is to find the Mighty Grimoire

But in the path you will find dreadful monsters that will try to stop you

Monsters will move one at a time in the direction of the hero

You lose the game if Dr.read's life total get's to 0

Combat rules

  • When a character (hero or monster) tries to move to another character's tile, they will attack the character in that tile
  • The attack damage is done by rolling a dice
  • If the defending character dies, the attacking character takes it's place
  • Monsters will also battle other monsters if they are in the way
